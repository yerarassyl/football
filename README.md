# Air Arena — бронирование футбольного поля

Одностраничный сервис заявок на аренду поля и административная панель. Проект
собран на Next.js и готов к размещению на Vercel.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`. Админ-панель находится по адресу
`http://localhost:3000/admin`.

Без `.env.local` приложение работает в демонстрационном режиме:

- логин: `admin`
- пароль: `admin123`
- вместо Google Sheets используются тестовые заявки

## Подключение Google Sheets

1. Создайте таблицу Google Sheets.
2. В Google Cloud создайте Service Account и включите Google Sheets API.
3. Откройте доступ к таблице для email сервисного аккаунта с правами редактора.
4. Скопируйте `.env.example` в `.env.local` и заполните переменные.
5. Создайте лист `Брони`. Заголовки и порядок колонок указаны в `lib/sheets.ts`.

Переменные для Vercel добавляются в `Project Settings → Environment Variables`.
Значение `GOOGLE_PRIVATE_KEY` должно содержать переносы строк в виде `\n`.

## Деплой на Vercel

Загрузите проект в GitHub, импортируйте репозиторий в Vercel и добавьте те же
переменные окружения. Команда сборки: `npm run build`.

## Что реализовано

- выбор 1/4, 1/2 или полного поля;
- цены, дата, время и визуальная индикация занятых слотов;
- выбор доступного сектора;
- контактная форма и отправка заявки;
- защищённый вход администратора;
- список заявок и статусы оплаты;
- предоплата, способ оплаты и комментарий;
- ручное создание и отмена брони;
- серверная интеграция с Google Sheets.
